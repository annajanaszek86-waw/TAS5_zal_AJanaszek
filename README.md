# TAS5 Java Selenium

Java version of the Python Selenium tests from:
`annajanaszek86-waw/TAS5_AJanaszek_praca_zal_python_selenium`.

## Requirements

- Java 17 or newer
- Maven
- Google Chrome

## Run in Visual Studio Code

1. Open this folder in VS Code.
2. Install extensions: `Extension Pack for Java` and `Maven for Java`.
3. Let VS Code import the Maven project.
4. Run tests from the Testing panel or terminal:

```powershell
mvn test
```

Screenshots are saved to `screenshots/`.
